rtou
====

.. _rtou:

**Description**

Maps an input from range {A to B} to unit range {0 to 1}

**Inputs, output and other parts**

*Input* (stream input) Value to be mapped.

*A* (stream input) Input range lower value.

*B* (stream input) Input range upper value.
