midi_to_freq
============

.. _midi_to_freq:

**Description**

Converts MIDI note number (0-127) to correnspoding frequency (Hz). E.g. 69 is A5 (440Hz) and 60 is C4 (261.63Hz).

**Inputs, output and other parts**

*MIDI note* (stream input) MIDI note value (0-127) to convert.
